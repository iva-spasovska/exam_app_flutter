# Exam App (Flutter)

A Flutter app to view exams and track remaining time for each exam.

---

## Features
- View all exams for a student index in a scrollable list.
- Detailed view for each exam (name, date, time, room, status, remaining time).
- Automatically calculates remaining time or shows if the exam is finished.
- Shows total number of exams at the bottom.
